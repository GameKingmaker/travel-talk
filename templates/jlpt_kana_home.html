📌 JLPT 기초(가나) — “처음 보는 사람”도 읽게 만들기

이 페이지는 일본어 기초 학습을 위한 히라가나 · 가타카나 완벽 정리와 JLPT 입문자를 위한 읽기 규칙을 한 번에 익힐 수 있는 공간입니다.

히라가나 표와 가타카나 표를 통해 일본어 글자를 빠르게 익히고, 촉음(っ), 장음(ー), 요음(ゃゅょ), 탁음/반탁음 등 일본어 읽기 필수 규칙을 쉽게 이해할 수 있습니다.

일본어를 처음 시작하는 초보자부터 JLPT N5 준비까지 가장 빠르게 일본어를 읽을 수 있도록 구성된 기초 학습 페이지입니다.

✅ 목표: 읽기 속도 올리기
✅ 핵심: 규칙 4개만 기억
✅ 추천: 매일 3분
학습 방법 읽는 규칙 히라가나 가타카나 추가 규칙 정리
맨위 학습 가이드 규칙 4종 히라가나 표 가타카나 표 추가 규칙 정리

🧠 “왜 글자가 두 개(히라/가타)예요?”

소리는 같지만 **쓰임이 달라서** 글자도 두 가지예요. 처음에는 “쓰임”은 몰라도 되고, **소리만 정확히** 익히면 됩니다.

  • 히라가나: 기본 단어/조사/문법에 많이 사용
  • 가타카나: 외래어/강조/이름표기 등에 자주 사용
Tip: 먼저 “소리(읽기)”만 잡으면, 쓰임은 자연스럽게 따라와요.

📈 초보자용 “가장 빠른 학습 루트”

하루 3~5분으로도 충분해요. 핵심은 “기억”보다 **반복 노출**입니다.

  • 1단계: 표를 한 줄(5글자)씩 눈으로 읽기
  • 2단계: い/う/え 같은 가운데 열부터 속도 올리기
  • 3단계: 아래 규칙 4개(촉음/장음/요음/탁음)만 익히기
  • 4단계: 단어에서 “규칙 찾기 게임”처럼 적용하기
Tip: “천천히 정확하게 → 점점 빠르게”가 정석이에요.

📌 읽는 규칙 (이 4개만 알면 갑자기 읽혀요)

아래 예시를 “그대로 소리내기”만 해도 효과가 커요.

1) 작은 っ (촉음) 잠깐 멈춤

작은 っ는 소리를 “꽉!” 막고 다음 자음을 강하게 시작해요.

예) がっこう → 가 (멈춤)코오
예) きっぷ → 킷푸
예) さっき → 삿키
Tip: ‘멈춤’ 자체가 소리예요. (가-코오가 아니라 가(멈춤)코오)

2) 장음 (길게) 길~게

소리를 길게 늘려요. 가타카나는 보통 ー로 표시합니다.

예) コーヒー → 코—히—
예) スーパー → 스—파—
예) おかあさん → 오카아상(あ가 늘어남)
Tip: 장음은 “글자 1개 더” 있다고 생각하면 쉬워요.

3) 요음 (ゃ/ゅ/ょ) 합쳐서 1소리

큰 글자 + 작은 ゃ/ゅ/ょ는 “두 글자를 합쳐 한 소리”로 읽어요.

예) きゃ = き(키) + ゃ(야) → 캬
예) しゅ = し(시) + ゅ(유) → 슈
예) りょこう → 료코오(りょ=료)
Tip: 작은 ゃゅょ는 “야/유/요”가 아니라 “ㅑ/ㅠ/ㅛ” 느낌!

4) 탁음/반탁음 (゛゜) 소리 바뀜

같은 글자에 점(゛)이나 동그라미(゜)가 붙으면 소리가 바뀌어요.

탁음(゛): か→が(카→가), さ→ざ(사→자), た→だ(타→다)
반탁음(゜): は→ぱ(하→파), ひ→ぴ(히→피)
Tip: “점=더 무겁게(가/자/다/바)” “동그라미=파행(파/피/푸/페/포)”
✅ (추가 팁) ん(ん/ン)은 왜 “응”처럼 들리나요?
ん은 단독으로 “응/ㄴ” 느낌 소리예요. 다음 자음에 따라 살짝 달라지지만
초보 단계에서는 그냥 “응”으로 통일해도 충분합니다.

예) こんにちは → 콘니치와 / さんぽ → 산포 / しんぶん → 신분(가까운 느낌)

📌 추가 규칙 정리 (탁음/반탁음 · 소문자 · 요음)

히라/가타 표를 눈에 익힌 뒤, 아래 규칙만 알면 단어가 훨씬 빨리 읽혀요.

1) 탁음(゛) / 반탁음(゜)

같은 글자에 점(゛)이나 동그라미(゜)가 붙으면 소리가 바뀌어요.

が行: が=가 / ぎ=기 / ぐ=구 / げ=게 / ご=고
ざ行: ざ=자 / じ=지 / ず=즈 / ぜ=제 / ぞ=조
だ行: だ=다 / ぢ=지 / づ=즈 / で=데 / ど=도
ば行: ば=바 / び=비 / ぶ=부 / べ=베 / ぼ=보
ぱ行(반탁음): ぱ=파 / ぴ=피 / ぷ=푸 / ぺ=페 / ぽ=포
Tip: “점(゛)=더 무겁게(가/자/다/바)” “동그라미(゜)=파행(파/피/푸/페/포)”

2) 소문자(ぁぃぅぇぉ / ゃゅょ / ゎ) & 가타카나 소문자(ァィゥェォ / ャュョ / ヮ)

소문자는 “작게 쓰는 글자”로, 보통 요음(ゃゅょ)을 만들거나 발음을 조절할 때 사용돼요.

히라가나 소문자: ぁ=아, ぃ=이, ぅ=우, ぇ=에, ぉ=오
요음 소문자: ゃ=야, ゅ=유, ょ=요
작은 ゎ: ゎ=와
가타카나 소문자: ァ=아, ィ=이, ゥ=우, ェ=에, ォ=오
요음 소문자: ャ=야, ュ=유, ョ=요, ヮ=와
Tip: 작은 ゃゅょ는 “야/유/요”라기보다 “ㅑ/ㅠ/ㅛ” 느낌(합쳐서 한 소리)

3) 요음(きゃ/しゃ/ちゃ …) = “두 글자를 합쳐 한 소리”

큰 글자(き/し/ち/に/ひ/み/り…) + 작은 ゃゅょ 를 합쳐 읽어요.

그룹 ゃ ゅ ょ 예시 단어
きゃ/きゅ/きょ きゃ = 캬 きゅ = 큐 きょ = 쿄 きょう(쿄오), きゅう(큐우)
しゃ/しゅ/しょ しゃ = 샤 しゅ = 슈 しょ = 쇼 しゅくだい(슈쿠다이)
ちゃ/ちゅ/ちょ ちゃ = 챠 ちゅ = 츄 ちょ = 쵸 ちょっと(춋토: 촉음 포함)
にゃ/にゅ/にょ にゃ = 냐 にゅ = 뉴 にょ = 뇨 にゅうがく(뉴우가쿠)
りゃ/りゅ/りょ りゃ = 랴 りゅ = 류 りょ = 료 りょこう(료코오)
✅ 가타카나 요음도 동일: キャ/キュ/キョ, シャ/シュ/ショ … (발음 규칙 똑같아요)
